# 8. 스프링 부트 소개

* <I>스프링 부트란</I>
* <I>스프링 부트 장점</I>

<br><br>

### 스프링 부트란

자바 기반으로 만들어진 독립 실행형 프레임워크로, 바로 실헁 가능한 프로덕션 급의 스프링 기반 어플리케이션을 쉽게 만들 수 있다. 

<br>

### 스프링 부트 장점

- 프로젝트 초기 세팅 과정이 단순해짐
- 프로젝트 진행 시 기본적으로 설정하게 되는 부분들은 부트가 지원하며 개별적으로 설정해야되는 부분만 설정 파일에 적용하면 됨 (예: DB 커넥션 설정 시 드라이버나 트랜잭션 등은 부트에서 자동으로 지원해 준다)
- 스프링 프레임워크 구성 요소들을 쉽게 가져다 사용할 수 있음 (예: Spring Data JPA, Spring Security 등)
- 톰캣이나 제티 등의 WAS를 내장할 수 있어서 프로젝트 동작 시간이 단축되며 서블릿 컨테이너가 내장될 수 있어서 프로젝트를 .jar 형식으로 간단하게 만들어 배포할 수 있음
